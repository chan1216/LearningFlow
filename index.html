Learning Flow

AI 기반 학습 자동화 플랫폼입니다.

PDF와 이미지 기반 학습 자료를 AI가 자동으로 요약하고, 퀴즈와 오답노트를 생성하여 학습 효율을 극대화하는 플랫폼입니다.

📌 기획 배경

📍 문제 인식

  • 많은 학습자들이 강의나 문서를 정리하는 데 과도한 시간을 사용하고 있습니다.
  • 요약, 정리, 오답 관리와 같은 반복적인 작업이 학습 효율을 낮추는 주요 원인입니다.
  • 기존 요약 도구는 단순 요약 기능에 머물러, 학습 과정 전체를 지원하지 못합니다.

🧠 인사이트

  • 실제 학습자는 “요약 → 퀴즈 → 오답”의 흐름을 반복하며 학습 효과를 강화합니다.
  • 따라서 요약뿐 아니라 퀴즈 생성과 오답 자동 정리를 일괄적으로 처리하는 통합 학습 플랫폼의 필요성을 확인했습니다.
  • AI 모델을 활용하면 학습 자료의 핵심 정보를 정확하게 추출하고 개인 맞춤형 학습 경험을 제공할 수 있다고 판단했습니다.

📌 핵심 기능

🎯 AI 학습 요약

사용자가 업로드한 PDF, 이미지, 텍스트 자료를 AI가 핵심 문장 중심으로 자동 요약합니다.

📝 자동 퀴즈 생성

요약된 내용을 기반으로 AI가 주요 개념을 분석하여 객관식 또는 단답형 문제를 자동 생성합니다.

📚 오답노트 자동 관리

사용자가 틀린 문제를 자동으로 저장하고 날짜별 및 유형별로 정리하여 체계적인 복습을 지원합니다.

🔍 이미지 요약 기능

사진과 스크린샷 등 이미지 파일에서 텍스트를 추출한 후 요약하여 다양한 학습 자료를 지원합니다.

🖥 PDF 변환 기능

요약 결과를 PDF로 변환하여 다운로드할 수 있으며 정리된 학습 자료로 활용할 수 있습니다.

🔄 전체 데이터 흐름

  1. User → Frontend: 사용자가 PDF 또는 이미지 파일을 업로드하면 전처리 후 서버로 전달합니다.
  2. Frontend → Backend (FastAPI): 업로드된 자료를 텍스트로 변환한 뒤 요약 요청을 서버로 전달합니다.
  3. Backend → OpenAI API: AI 모델을 통해 텍스트 요약, 퀴즈 생성, 오답 분석을 수행합니다.
  4. Backend → Database (SQLAlchemy): 요약본, 생성된 문제, 오답 정보, 학습 이력을 데이터베이스에 저장합니다.
  5. DB → Frontend: 요약본, 퀴즈, 오답노트 데이터를 UI로 반환하여 사용자에게 제공합니다.
